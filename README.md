## 🚨 The Problem

India ke government hospitals mein servers crash ho jaate hain — koi monitor nahi karta!

> 🔴 **AIIMS Delhi** — 2022 mein server hack hua. **23 din** tak sab manually karna pada.

| Without MediOps ❌ | With MediOps ✅ |
|-------------------|----------------|
| Server crash → nobody knows | Crash detected in **15 seconds** |
| Manual restart → 6+ hours | Auto-restart in **45 seconds** |
| Patient data lost | Data safe in **AWS S3** |
| No warning before crash | AI warns **2 hours** in advance |

## 🚀 Quick Start

```bash

cd mediops
docker-compose up -d
```
